SublimeCodeIntel 无法智能提示 解决方案

  • weichao
  • 2 Minutes
  • March 6, 2017

SublimeCodeIntel 是一个支持 一下如此多的语言代码智能提示的插件(温馨提示: 作者已经没持续更新了)

    /*
       An array of language names which are enabled.
    */
    "codeintel_enabled_languages":
    [
       "JavaScript", "SCSS", "Python", "HTML",
       "Ruby", "Python3", "XML", "Sass", "HTML5", "Perl", "CSS",
       "Twig", "Less", "Node.js", "TemplateToolkit", "PHP"
    ]

大概的意思是要你自己创建一个.codeintel 文件夹,然后在里面建个 config.log
输入

{
    "PHP": {
        "php": '/usr/bin/php',
        "phpExtraPaths": [],
        "phpConfigFile": 'php.ini'
    },
    "JavaScript": {
        "javascriptExtraPaths": []
    },
    "Perl": {
        "perl": "/usr/bin/perl",
        "perlExtraPaths": []
    },
    "Ruby": {
        "ruby": "/usr/bin/ruby",
        "rubyExtraPaths": []
    },
    "Python": {
        "python": '/usr/bin/python',
        "pythonExtraPaths": []
    },
    "Python3": {
        "python": '/usr/bin/python3',
        "pythonExtraPaths": []
    }
}

然后关闭 sublime text 重新打开就行了,然后你就会神奇的发现,OK了,智能提示已经出现啦,哈哈哈.

当然,你要知道,在window 中 是不允许创建 以 “.”开头的文件夹的,这时候你就要用到控制台了

mkdir ..codeintel

然后再去建一个config.log文件就行了.
